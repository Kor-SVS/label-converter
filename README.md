### .lab <-> .TextGrid 라벨 상호변환 프로그램

### 사용 방법

-   실행 파일에 폴더 또는 파일 드래그 드롭
-   실행 -> 폴더 선택

#### dict 지원

-   실행파일과 같은 위치에 `dict.txt`파일을 생성하고 다음과 같은 형식으로 작성합니다.
-   띄어쓰기가 있을 경우, 여러줄에 걸처 변환을 진행합니다.

```
변환전 음소=변환후 음소
```

(example)

```
w e=we
w i=wi
eu i=eui
a=au
e=eo
i=ui
...
```

### 빌드

```sh
go build -ldflags "-s -w" .
```
